# Local Market Online Store

 Project Overview
This is a frontend-based web application designed to connect local vendors with customers. It allows users to browse products easily through a clean and responsive interface.

## Features
- User-friendly interface
- Product listing using cards
- Login and registration UI
- Responsive design (mobile, tablet, desktop)

## Technologies Used
- HTML (structure)
- CSS (styling & layout)
- JavaScript (interactivity & DOM manipulation)

## Functionality
- DOM manipulation for dynamic content
- Event handling for user interaction
- Responsive design using Flexbox and media queries

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
